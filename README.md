# TODO LIST

APP is live at: [https://jovanailin.github.io/my-todo-list/]

This is a simple *Todo App* done with Vue and Vuetify.
It has the following functions: 

- Add New Task
- Mark Task as DONE
- Edit Task
- Clear specific tasks
- Clear DONE tasks
- Clear ALL tasks
